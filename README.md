## Projeto Big Data com Apache Spark

### Descrição
Projeto para análise de dados CSV utilizando Apache Spark.

### Pré-requisitos
- Python 3
- Apache Spark ou Databricks

### Execução
1. Criar a pasta `dados`
2. Colocar o arquivo CSV dentro dela
3. Executar os scripts da pasta `src`

### Observação
O arquivo CSV não é versionado no Git.
